Genetic prioritisation, therapeutic repositioning and cross-disease comparisons reveal inflammatory targets tractable for kidney stone disease

5.1 analyse using the supra-hexagonal map

# read cross-disease prioritisation matrix
data <- read_delim('outputs/crossdisease_matrix.txt', delim='\t') %>% column_to_rownames('target')

# trained using the self-organising learning algorithm
sMap <- data %>% sPipeline(scaling=1)

# the trained map partitioned into gene clusters
sBase <- sMap %>% sDmatCluster()

# write into a file 'crossdisease_suprahex.txt'
sWriteData(sMap, data, sBase, keep.data=T) %>% transmute(Target=ID, Index=str_c('H',Hexagon_index), Cluster=str_c('C',Cluster_base), KSD,AS,PSO,UC,CRO,RA,MS,SLE,T1D) %>% write_delim('outputs/crossdisease_suprahex.txt', delim='\t')
